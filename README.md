### 善逸教育-后端(开发中...)
前端部分，使用主流的前端框架Vue，使用Es6的开发规范，采用模块化的开发模式。
后端部分，使用目前流行的SpringBoot+SpringCloud进行微服务架构，使用Feign、Gateway、Hystrix，以及阿里巴巴的Nacos等组件搭建了项目的基础环境。
